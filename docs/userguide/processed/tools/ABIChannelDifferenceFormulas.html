Description of Formulas - ABI Channel Differences

The following describes each formula included in the ABI (Advanced Baseline Imager - GOES-16 through GOES-19) Channel Difference Formulas section of McIDAS-V. These formulas create displays of various channel differences that are useful to visualize different features. These are also derived formulas, meaning if a data source is loaded containing the necessary bands for a formula, they will be listed in the Field Selector under the ABI Channel Differences tree. For specifics about the usage of each of these channel differences, see the CIRA VISIT Quick Guides. Many of these formulas have parameter defaults defined, meaning a default enhancement and enhancement range will be used in the display.

  • ABI Night Fog Difference (Band 13T, Band 7T) - Identifies clouds made up of water droplets at night.
    Input Parameter Definition

    Band 13T

    Band 13 temperature.

    Band 7T

    Band 7 temperature
  • ABI Split Cloud Phase Difference (Band 14T, Band11T) - Differentiates between types of clouds. For example, thick and thin cirrus, and clouds made of ice and water.
    Input Parameter Definition

    Band 14T

    Band 14 temperature.

    Band 11T

    Band 11 temperature.
  • ABI Split Ozone Difference (Band 12T, Band 13T) - Displays ozone absorption compared to the clean window.
    Input Parameter Definition

    Band 12T

    Band 12 temperature.

    Band 13T

    Band 13 temperature.
  • ABI Split Snow Difference (Band 5B, Band 2B) - Emphasizes regions where ice is present, including glaciated clouds and snow/ice.
    Input Parameter Definition

    Band 5B

    Band 5 brightness.

    Band 2B

    Band 2 brightness.
  • ABI Split Water Vapor Difference (Band 12T, Band 13T) - Displays water vapor concentration and distribution.
    Input Parameter Definition

    Band 12T

    Band 12 temperature.

    Band 13T

    Band 13 temperature.
  • ABI Split Window Difference (Band 15T, Band 13T) - Emphasizes low level moisture and dust.
    Input Parameter Definition

    Band 15T

    Band 15 temperature.

    Band 13T

    Band 13 temperature.
