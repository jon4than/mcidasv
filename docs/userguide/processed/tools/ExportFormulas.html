Description of Formulas - Export

The following describes each formula included in the Export Formulas section of McIDAS-V.

Note: Both of these formulas utilize a Select Input window to enter a file name to be used by the formula. You can choose if you want to save this file name in a bundle if you wish to create one. The file name field of the Select Input window has a Save in Bundle option (chosen by default). If you keep this option selected, the file name you choose will be stored in the bundle, and the formula will automatically execute when you load in your bundle without any user input. If you do not save the values in your bundle, the file name you use will not be saved, and the formula will not automatically execute without requiring the user to enter/change the file name field.

  • Export Grid to Excel (Excel File) - Exports a field to an Excel (.xls) file and creates a display of the field.
  • Export Grid/Image to netCDF (netCDF File) - Exports a grid or image field to a netCDF (.nc) file and creates a display of the field.
