Description of Formulas - AHI RGBs

The following describes each formula included in the AHI RGB Formulas section of McIDAS-V.

  • AHI Airmass RGB (name, unitSpec) - Sets the name and unit of a field.
    Input Parameter Definition

    name

    The name of the new unit.

    unitSpec

    The unit that will be used in the display of your parameter.  This must be a unit recognized by McIDAS-V, and it must be convertible with the default unit of your data.
