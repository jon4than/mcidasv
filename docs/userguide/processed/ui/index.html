Main Display Window
The Main Display window within McIDAS-V contains:
  • A 3D view of the displays.
  • A Legend panel that shows information about the current displays.
  • A main menu bar and main toolbar that allows for controlling the application.
  • A subsidiary view menu bar that is relevant to the 3D display.
  • Time animation display and control widgets.
  • Navigation toolbars.

This section describes:

  • Menu Bar
  • Main Toolbar
  • Drag and Drop Tabs
  • Display Menus
  • Properties Dialog
  • Visibility Animation
  • Time Animation
  • Legend
  • Modify the Color Bar Interactively
  • Status Bar
  • Zooming, Panning and Rotating
  • Transect Views
